# Guts and Blackpowder Wiki

> The complete community wiki for [Guts & Blackpowder](https://gutsandblackpowder.wiki) — a Roblox PvE zombie survival game set during the Napoleonic Wars. 533M+ plays, 94% rating, 540K+ Discord members.

## 📖 About

Guts and Blackpowder is a free-to-play Roblox horde shooter developed by Fuze Studios, released August 2021. Set in an alternate-history Napoleonic era where "The Blight" has raised the dead across Europe, players fight through 15 historical maps armed with muskets, sabres, and 19th-century grit.

This wiki covers every aspect of the game, maintained by the community.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Deployment:** [Vercel](https://vercel.com)
- **Styling:** Tailwind CSS
- **Analytics:** Google Analytics 4
- **Domain:** gutsandblackpowder.wiki (Cloudflare)

## 🚀 Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## 📦 Build

```bash
pnpm build
```

Static export to `out/`, deployed via Vercel with `cleanUrls`.

## 🤝 Contributing

This is a community-maintained wiki. If you spot inaccurate info, missing content, or want to add a guide:

1. Fork the repo
2. Create a branch (`git checkout -b fix/weapon-stats`)
3. Commit your changes
4. Open a PR

All game data lives in `src/lib/data.ts` — that's where most content updates go.
