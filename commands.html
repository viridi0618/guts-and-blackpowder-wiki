Getting Started

How to Use Commands

Requirements and basic usage

You must be a private server owner, game admin, or developer to use commands. The easiest way to get access is to create a private server.

Steps to use a command:

  1. Open the Roblox in-game chat
  2. Type / followed by the command (e.g. /give @m Musket)
  3. Press Enter to activate

If chat is blocked in your region or your account is not age-verified, press ] on keyboard (PC) or the left D-pad button on console to open the Command Bar instead.

Syntax

Target Arguments (@a / @m / @o)

How to target players with commands

Most commands require a [user] target. Instead of typing a player's name, use these shortcuts:

ArgumentEffect
@aAffects all players on the server, including yourself
@mAffects only yourself
@oAffects all other players, excluding yourself

Example: /give @a Musket gives every player a Musket at once.

FAQ

Frequently Asked Questions

Common questions about using commands

Can I use commands without a private server?

No. Commands require admin permissions, which regular players only get in private servers they own.

Can commands be used on console / mobile?

Yes. On console, press the left D-pad button to open the Command Bar. On mobile, use the on-screen chat button and type the command normally.

Why isn't my command working?

Check that you are the private server owner, that you typed the / prefix, and that the player name or @ argument is correct. Some commands like /arthur have timing restrictions.

What's the fastest way to get Francs using commands?

Use /hardcore to double all Franc rewards, then play Endless mode to farm waves efficiently. See our Francs guide for more strategies.

Can I spawn the Headless Horseman or Dracula with commands?

No — these boss enemies are tied to the Sleepy Hollow and Transylvania seasonal maps and cannot be manually spawned.
